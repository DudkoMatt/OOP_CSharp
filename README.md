# OOP CSharp
В данном репозитории находятся лабораторные работы выполненные на C# согласно принципам ООП и популярным [паттернам проектирования](https://refactoring.guru/ru/design-patterns).

## Лабораторные работы

1. [INI файл](Lab_1/OOP1.pdf) - создать инструмент для обработки конфигурационного INI файла;
2. [Магазин](Lab_2/OOP2.pdf) - создать библиотеку для организации работы магазинов - магазин, товары;
3. [Симулятор гонок](Lab_3/OOP3.pdf) - разработать примитивный движок для фэнтезийного симулятора гонок;
4. [Бекапы](Lab_4/OOP4.pdf) - разработать систему контроля версий файлов (аналог git);
5. [Банки](Lab_5/OOP5.pdf) - разработать библиотеку для организации работы банка - несколько типов счетов, клиенты, банки;
6. [Отчёты](Lab_6/OOP6.pdf) - реализовать механизм автоматизации создания отчетов о проделанной командной работе за спринт, используя многослойную архитектуру.

Код лабораторных работ можно найти в папках с соответствующими номерами. Для удобства написания "хода рассуждений" был создан bash-скрипт, генерирующий файл со всеми именами исходных файлов.
